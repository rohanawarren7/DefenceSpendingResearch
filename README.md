# DefenceSpendingResearch

Our world is changing rapidly, however we're seeing a shift in global power in this moment which will likely be reflected on as a stark inflection point in geopolitcal history. 
Through the apparent chaos in various areas, my aim is to see if we can see order in the chaos. 

As such, I want to explore the political and militarial landscape to find trends that could be used for further research, which could be useful for deriving insights in both finding 
opportunities for investment purposes or develop knowledge within academia.

Areas of Interest
*Can the sentiment trajectory of U.S. National Security Strategy/NDS documents predict the direction of force posture changes 3–5 years ahead?*
I will be applying NLP sentiment analysis and topic modeling. The literature confirms this methodolgy is viable: transformer-based models (FinBERT derivatives, RoBERTa) are already used for geopolitical media sentiment, and military applications of LLMs explicitly include sentiment analysis for information operations.
